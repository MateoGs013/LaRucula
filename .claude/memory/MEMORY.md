# LaRucula — Long-Term Memory

## Project summary
LaRucula is a premium beachfront restaurant website for Spain.
The site should feel editorial, warm, artistic, Mediterranean, and refined.

## Core stack
- Vue 3
- Vite
- Tailwind CSS
- GSAP
- Vue Router
- SVG motion accents

## Core design principles
- Avoid generic restaurant layouts
- Favor editorial asymmetry and whitespace
- Use serif display typography with restraint
- Keep handwritten accents subtle
- Motion should support atmosphere and hierarchy

## Working model
- Codex handles project setup, scaffolding, architecture, and reusable foundations
- Claude handles front-end implementation, motion, SVG accents, and visual polish

## Scaffold status
- base Vue/Vite/Tailwind project scaffolded
- route placeholders exist for home, menu, story, visit, and not-found
- layout shells, token files, and GSAP helpers are ready for Claude to extend
- future implementation should build on the scaffold rather than replace it
