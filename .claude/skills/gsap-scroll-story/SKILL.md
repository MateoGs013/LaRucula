# Skill: GSAP Scroll Story

## Goal
Create a refined scroll-based motion sequence for LaRucula.

## Allowed patterns
- line reveal
- image mask reveal
- staggered text
- small scale drift
- pinned storytelling only if justified
- SVG path drawing

## Avoid
- excessive pinning
- loud visual tricks
- long blocked scroll moments
- unnecessary complexity

## Deliverable
- clean implementation
- reduced-motion fallback
- concise notes about the motion purpose
