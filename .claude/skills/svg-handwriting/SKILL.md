# Skill: SVG Handwriting Accent

## Goal
Create or refine subtle handwritten SVG accents for LaRucula.

## Visual rules
- premium
- controlled
- organic
- minimal
- editorial

## Use cases
- underline
- side note flourish
- separator stroke
- signature-like CTA support
- path drawing during scroll

## Avoid
- childish doodles
- heavy illustration
- overly decorative marks
