# Skill: Scaffold Editorial Section

## Goal
Create a new reusable editorial Vue section for LaRucula.

## Inputs
- section name
- page
- purpose
- content blocks
- motion complexity

## Steps
1. Read:
   - docs/brand.md
   - docs/ui-rules.md
   - docs/motion-rules.md
2. Inspect existing components in `src/components/`
3. Create a reusable section component
4. Use semantic HTML
5. Add clean responsive structure
6. Leave clear hooks for motion
7. Keep styling aligned with the design system
8. If a meaningful reusable pattern is introduced, update `docs/decisions.md`

## Output
- new component
- optional usage example
- note about possible animation hooks
