# Skill: Setup GSAP Motion

## Goal
Attach elegant GSAP motion to a section without polluting component structure.

## Rules
- prefer composables or dedicated motion helpers
- clean up on unmount
- respect reduced motion
- keep timelines modular
- favor subtle reveal, masking, drift, and line drawing

## Steps
1. Inspect the section structure
2. Identify primary and secondary motion targets
3. Implement GSAP / ScrollTrigger logic
4. Add cleanup
5. Ensure motion supports hierarchy and readability
